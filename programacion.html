Programación

Programación

Proceso de diseñar, escribir, probar y mantener el código fuente de software de computadora.

Este proceso implica el uso de un lenguaje de programación para crear instrucciones que una computadora puede seguir para realizar tareas específicas.

Lenguajes de Programación

Sistema formal de comunicación diseñado para expresar instrucciones que pueden ser comprendidas y ejecutadas por una computadora.

Lenguajes de Bajo Nivel

Estos lenguajes están más cerca del lenguaje máquina y son específicos del hardware. Son menos abstractos y más difíciles de leer para los humanos.

  • Ensamblador: Un lenguaje de programación de bajo nivel que utiliza abreviaciones simbólicas en lugar de códigos binarios.

Lenguajes de Medio Nivel

Estos lenguajes tienen una mezcla de características de bajo y alto nivel. Son más abstractos que los lenguajes de bajo nivel, pero aún permiten un control directo del hardware.

  • C: Un lenguaje de programación de propósito general que se utiliza tanto para sistemas operativos como para aplicaciones.

Lenguajes de Alto Nivel

Estos lenguajes son más abstractos y más fáciles de leer y escribir para los humanos. Son independientes del hardware.

  • Java: Un lenguaje de programación orientado a objetos que es ampliamente utilizado en aplicaciones empresariales.

Ejemplo de Código Fuente en Java

A continuación se muestra un ejemplo del método principal en Java:


public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, mundo");
    }
}
