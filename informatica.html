Componentes de una computadora

Hay computadoras de varios tipos, pero en este capítulo nos centraremos en el concepto general (computadora de escritorio).

Comenzaremos por las partes físicas, es decir, su hardware.

Dispositivos que permiten la interacción con la computadora:

  • Dispositivos de entrada:
    1. Teclado
    2. Ratón
    3. Micrófono
  • Dispositivos de salida:
    1. Monitor
    2. Impresora
    3. Altavoces

Estructura Interna de una Computadora

Unidad Central de Procesamiento (CPU):

La CPU es el cerebro de la computadora. Realiza todas las operaciones aritméticas, lógicas y de control de las instrucciones del programa.

Componentes principales:

  • Unidad de Control (CU): Dirige todas las operaciones dentro de la CPU y coordina las actividades de los otros componentes.
  • Unidad Aritmético-Lógica (ALU): Realiza operaciones aritméticas y lógicas.
  • Registros: Pequeñas áreas de almacenamiento rápido dentro de la CPU para datos temporales y de control.

Unidad de Procesamiento Gráfico (GPU):

Procesador especializado para manejar gráficos y video.

Es crucial para tareas intensivas como juegos, diseño gráfico y modelado 3D.

Memoria RAM:

La memoria de acceso aleatorio (RAM) es la memoria temporal que la CPU usa para almacenar datos y programas que están en uso. Es volátil, lo que significa que pierde su contenido al apagar la computadora.

Características:

  • Alta velocidad
  • Acceso directo
  • Capacidad variable

Unidad de almacenamiento:

Almacenamiento permanente para datos y programas. Tipos:

  • Disco Duro (HDD): Almacenamiento magnético con grandes capacidades.
  • Unidad de Estado Sólido (SSD): Más rápido que los HDD y sin partes mecánicas.

Motherboard:

Placa principal que conecta todos los componentes de la computadora.

Componentes:

  • Socket: Conector para la CPU.
  • Ranura de Memoria: Para instalar RAM.
  • Chipset: Controla la comunicación entre la CPU, la memoria y otros componentes.
  • Ranura de expansión: Permite conectar tarjetas adicionales.
  • ROM: Memoria de solo lectura que se usa para almacenar programas constantes que no están sujetas a cambios.

Esta es una visión general de los componentes internos de una computadora y sus funciones.

Cada uno de estos componentes juega un papel crucial en el funcionamiento de una computadora. Sin embargo, además de estos, la computadora necesita más elementos para poder organizar sus tareas, y es aquí donde entra el software, que veremos a continuación.

Software de una computadora:

El software se clasifica según su función.

Software de Aplicación:

Diseñados para realizar tareas específicas. Ejemplos:

  • Procesador de texto
  • Herramientas de seguridad
  • Exploradores

Software de Desarrollo:

Herramientas que ayudan a crear software. Ejemplos:

  • Editores de código
  • Compiladores
  • Depuradores
  • Entornos de desarrollo integrado

Software de Sistema:

Programas que permiten la gestión del hardware y proporcionan servicios básicos para otras aplicaciones. Ejemplos:

  • Firmware
  • Controladores
  • Sistema operativo
