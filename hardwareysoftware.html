Hardware Y Software

Hardware

El hardware de la computadora es el circuito electrónico que realiza el trabajo real.

El hardware incluye elementos con los que ya está familiarizado, como el procesador, la memoria, el teclado, la grabadora de CD, etc.

Fundamentals of Computer Organization and Design
De Sivarama P. Dandamudi

Software

El software es el conjunto de programas que indican a la computadora las tareas que debe realizar.

Estos programas controlan y dirigen a la computadora para que realice un conjunto de acciones (instrucciones).
