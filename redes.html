Redes

Redes de Computadoras

Una red de computadoras comprende dos o más computadoras conectadas, ya sea por cables o inalámbrico, con el propósito de transmitir, intercambiar o compartir datos y/o recursos.

Se construye una red informática utilizando hardware, como módems, puntos de acceso y cables, y software, como sistemas operativos o aplicaciones comerciales.

Tipos de redes de computadoras

  • WAN (red de área amplia): Una WAN conecta computadoras en un área amplia, como de una región a otra o incluso de un continente a otro.
  • MAN (red de área metropolitana): Las MAN suelen ser más grandes que las LAN, pero más pequeñas que las WAN.
  • LAN (red de área local): Una LAN conecta equipos a una distancia relativamente corta, lo que les permite compartir datos, archivos y recursos.
  • WLAN (red de área local inalámbrica): Una WLAN es como una LAN, pero las conexiones entre dispositivos en la red se realizan de forma inalámbrica.

Conceptos importantes

Nodos: Esencialmente cualquier dispositivo de red que pueda reconocer, procesar y transmitir información. Algunos ejemplos de nodos incluyen computadoras, impresoras, módems, etc.

Enrutador: Dispositivo físico o virtual que envía información contenida en paquetes de datos entre redes.

Puertos: Un puerto identifica una conexión específica entre dispositivos de red. Cada puerto se identifica con un número.

Cables de red: Los tipos de cables de red más comunes son:

  • UTP
  • Coaxial
  • Fibra óptica

Dirección IP: Una dirección IP es un número único asignado a cada dispositivo conectado a una red.

¿Cómo funcionan?

Las redes de computadoras conectan nodos mediante cables o señales inalámbricas.

Las redes siguen protocolos, que definen cómo se envían y reciben las comunicaciones.

Cada dispositivo en una red utiliza una IP que identifica de forma única un dispositivo y permite que otros lo reconozcan.
